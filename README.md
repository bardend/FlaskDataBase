# Db_Service



1. Clona el repositorio:

    ```bash
    git clone https://github.com/bardend/FlaskDataBase.git
    ```

2. Navega al directorio app, instala las dependencias:

    ```bash
    cd app
    pip install -r requirements.txt
    ```
    
3. Ejecuta el docker-compose:

    ```bash
    cd ..    
    docker-compose up 
    ```
4. Ejecutar el test:

    ```bash
    cd test    
    python3 cliente2.py 
    ```
